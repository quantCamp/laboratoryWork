## Теоретическая часть

### Случайность и вероятность
Если мы подбросим обычную честную монетку, у нас есть два возможных исхода: **Орел** или **Решка**. Так как монетка симметрична, шансы выпадения обеих сторон одинаковы. 

В математике вероятность события измеряется от 0 до 1 (или от 0% до 100%). Вероятность выпадения орла равна:
$$P = 0.5 \quad (50\%)$$

Однако, если вы подбросите монетку всего 10 раз, вы вряд ли получите ровно 5 орлов и 5 решек. У вас может выпасть 7 орлов и 3 решки, или даже 9 орлов и 1 решка. Это происходит из-за случайных колебаний.

### Закон больших чисел
**Закон больших чисел** — это правило в теории вероятностей, которое гласит: по мере увеличения количества независимых испытаний (в нашем случае — бросков монеты), средний результат приближается к теоретическому ожиданию.

Иными словами: чем больше раз мы бросаем монетку, тем ближе доля выпавших орлов будет к $0.5$ (то есть к 50%). Если при 10 бросках мы можем получить 70% орлов, то при 100 000 бросков мы почти наверняка получим очень близкое к 50% значение (например, 50.1% или 49.9%).

В этой лабораторной работе мы проверим этот закон с помощью компьютерного моделирования!

### Моделирование броска монеты в Python
Чтобы сымитировать бросок монеты, мы можем использовать функцию `randint(a, b)` из встроенного модуля `random`. Она возвращает случайное целое число от `a` до `b` включительно.

Давайте договоримся, что:
* `0` — это выпадение **Решки**
* `1` — это выпадение **Орла**

Пример кода:
```python
import random

# Имитируем один бросок монеты
brosok = random.randint(0, 1)

if brosok == 1:
    print("Выпал Орел!")
else:
    print("Выпала Решка!")
```

### Замер времени работы кода
Компьютер выполняет операции очень быстро, но не мгновенно. Чтобы узнать, сколько времени тратит процессор на выполнение циклов, мы используем модуль `time` и функцию `time.perf_counter()`. Она измеряет время в секундах с высокой точностью.

Шаблон для замера времени:
```python
import time

start = time.perf_counter() # Запоминаем время начала

# Какой-то код, например, цикл
for i in range(1000000):
    x = 2 + 2

end = time.perf_counter() # Запоминаем время окончания
elapsed = end - start     # Вычисляем разницу
print(f"Код выполнялся {elapsed:.6f} секунд")
```

### Задание 2. Функция для одного эксперимента с замером времени
Напишите функцию `run_single_experiment(n)`, которая будет запускать симуляцию, измерять время её работы, вычислять частоту выпадения орла и её отклонение от идеальных 50% ($0.5$).

Функция должна возвращать три значения:
1. Частоту выпадения орла (количество орлов разделить на общее число бросков `n`).
2. Абсолютное отклонение частоты от $0.5$ (вычисляется как модуль разности: `abs(частота - 0.5)`).
3. Время работы функции `flip_coin(n)` в секундах.

*Примерный шаблон кода:*
```python
import time

def run_single_experiment(n):
    start_time = time.perf_counter()
    
    # 1. Вызываем функцию flip_coin(n) и сохраняем результат
    heads = flip_coin(n)
    
    end_time = time.perf_counter()
    
    # 2. Вычисляем время работы
    elapsed_time = end_time - start_time
    
    # 3. Вычисляем частоту выпадения орла
    frequency = heads / n
    
    # 4. Вычисляем отклонение от 0.5
    deviation = abs(frequency - 0.5)
    
    # Возвращаем все три значения
    return frequency, deviation, elapsed_time
```
